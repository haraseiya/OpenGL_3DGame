【操作方法】
　※PAD操作
　左スティック　→　移動
　右スティック　→　視点移動
　Rボタン	→　弾発射
　Lボタン	→　ダッシュ
　Aボタン	→　シーン遷移
　ESCキー	→　ゲーム終了

【ジャンル】
　３D弾幕フロアシューティング

【全体的な流れ】
　目標：ボスの討伐
　流れ：敵を倒すと経験値アイテムをドロップし、一定数拾うごとにプレイヤー弾幕が強化される
　　　　倒した敵の数、種類に応じスコアが加算される

【使用ツール】
　・VisualStudio2019
　・Blender

【担当箇所】
　・個人製作

【使用言語】
　・C++

【製作期間】
　・5か月

【実装した機能】
　・処理負荷軽減用のジオメトリインスタンシング
　・処理負荷軽減用のオブジェクトプール
　・シャドウマップ、ブルーム、ポストエフェクトなどのGLSLを利用したシェーダー機能
　・コンポーネントパターンを採用したシステム設計
　・ステートパターンを採用したアニメーション遷移
　・エフェクシア読み込み機能
　・Mixamoのアニメーション読み込み機能

【作成リソース】
　・蜘蛛糸のモデル（Blender）
　・プレイヤーの複数テクスチャを1枚にまとめる（Blender）←軽量化のため

【exeファイルの位置】
　exe→gameProto.exe

【ソリューションファイルの位置】
　SpiderSwarm→gameProto.sln
　
【補足】
「アンドロイドカクタス」が参考作品の企業プロジェクトであり、敵と弾が大量に出てもフレームレートを維持できるようにするという課題の元制作を行った。
その他、シェーダーやキューブマップといった、OpenGLの機能を利用したグラフィックスプログラムを実装した。
